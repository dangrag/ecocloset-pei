# EcoCloset - Projeto PEI

## Overview
Sistema de controle de estoque com provador virtual focado em sustentabilidade. Projeto acadêmico PEI (Projeto de Extensão e Integração) para Análise e Desenvolvimento de Sistemas.

## Technical Approach
- All data stored in localStorage (no database)
- Base64 image storage for user-uploaded product photos
- Canvas-based virtual fitting room with drag-and-drop positioning
- Seed data auto-loaded on first visit (8 sample eco-friendly products)

## Visual Style
- **Theme**: Green/emerald eco-friendly palette
- **Primary**: HSL 152 69% 31% (forest green)
- **Accent**: HSL 80 60% 45% (lime green)
- **Default mode**: Light theme
- **Fonts**: DM Sans (body), Plus Jakarta Sans (display), JetBrains Mono (numbers)

## Page Structure
1. **/** - Landing page with hero, features, stats counters, featured products, about PEI section
2. **/catalogo** - Product catalog with search, filters (category, material, local suppliers, sort), add to cart
3. **/admin** - Stock management with CRUD operations, image upload, sustainability scoring
4. **/provador** - Virtual fitting room with photo upload, clothing overlay, drag/resize/opacity controls
5. **/carrinho** - Shopping cart with carbon footprint calculator, savings comparison vs fast fashion

## Key Features
- Sustainability index (1-10 score per material type)
- Local supplier badge system
- Carbon footprint calculation (CO2 per garment vs conventional fashion)
- Virtual fitting with Canvas overlay technology
- Auto-seeding with 8 demo products

## Components
- `components/eco/header.tsx` - Sticky nav with cart badge
- `components/eco/footer.tsx` - Footer with PEI info
- `components/eco/sustainability-badge.tsx` - Score badge + local supplier badge
- `components/eco/carbon-meter.tsx` - Carbon footprint visualization

## Data Models (lib/types.ts)
- Product: id, name, description, image, size, quantity, price, supplier, supplierCity, isLocal, material, sustainabilityScore, carbonFootprint, category
- CartItem: product, quantity
- MATERIALS map with score and carbon footprint per material
